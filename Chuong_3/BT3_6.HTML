CSS CĂN BẢN

1. CSS là gì?

Trong lĩnh vực xây dựng, chúng ta có trang trí nội thất; trong lĩnh vực thẩm mỹ - làm đẹp, chúng ta có kỹ thuật make-up; còn trong lĩnh vực thiết kế web chúng ta có CSS. CSS (Cascading Style Sheet) cho phép định dạng các tài liệu viết bằng HTML, XHTML, XML, SVG, UML,...

2. Học CSS cần những gì?

  • Kiến thức căn bản về HTML
  • Một trình soạn thảo CSS
  • Trình duyệt
  • Thời gian thực hành CSS

3. Những lệnh thường gặp trong CSS

  • background: background-color, background-image, background-position
  • font: font-family, font-style, font-weight, font-size
  • color
